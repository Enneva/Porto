Valorat Agent App

Simple Android Application build in Jetpack Compose, showing description, role, also Skill for each Agent on Valorant using API from Valorant API and Retrofit Library for consume the API

Skin Cancer Detector

An android application made to complete a class at Dicoding, this application has a skin cancer classification feature based on images, using a tensorflow model that has been converted to tensorflow lite. another feature is that it can display cancer-related news using NewsAPI and Retrofit library.

Storytopia

Project created to fulfill classes in Dicoding, this application has several features, namely login and create an account, users can create posts using images. this application also uses the Google Maps API which is useful for finding posts based on the user's area.

Sebatik

This application is a capstone project that I developed with the team at Bangkit Academy, the main feature of this application is image classification, the application can determine the name of batik based on its motif, and also provide additional information such as history and how to make the batik. This app also using Firebase for Login and Create Account, and Firebase Storage for storing image
